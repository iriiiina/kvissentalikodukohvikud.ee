## AI-assisted development

This project is set up for AI-assisted development using [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

* **CLAUDE.md** — a project instruction file in the repository root that provides Claude Code with context about the project: tech stack, file structure, design conventions, code style, and workflow rules. Claude Code reads this file automatically at the start of each session, so it can work with the codebase without lengthy onboarding.
* **No build tools needed** — since this is a pure static site, Claude Code can read and edit files directly without any setup or compilation step.
